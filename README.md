# Détection d'émotions avec furhat

Ce projet reprend différents projet de reconnaissance d'émotions (happy - fear - sad - disgust - suprise) avec furhat. Il a pour but de repérer les émotions réalisées par l'utilisateur devant la caméra intégrer du robot. Celui-ci doit pouvoir énoncer quelle émotion a été réalisé et la reproduire. Le modèle initial fine tuné est DenseNet-121
https://docs.furhat.io/remote-api/.
